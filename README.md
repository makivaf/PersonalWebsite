# 🌐 Personal Website

A responsive personal website developed using HTML and CSS to showcase my profile, projects, and front-end development skills. This personal website was originally created approximately three years ago as part of my early learning experience in front-end development. Some design choices, code structure, and features may not reflect my current skills and best practices, but it remains a valuable representation of my progress and growth over time.


## 🚀 Live Demo

👉 https://makivaf.github.io/PersonalWebsite/

## 📌 Overview

This project is a static personal portfolio website designed to present my background, skills, and projects in a clean and user-friendly interface. It focuses on simplicity, responsiveness, and structured layout.

## ✨ Features

* 📱 Responsive design for desktop and mobile devices
* 🎨 Clean and modern user interface
* 🧭 Easy navigation across sections
* ⚡ Fast loading (pure HTML & CSS, no frameworks)

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 🎯 What I Learned

* Structuring a complete webpage using semantic HTML
* Designing layouts using CSS (flexbox/grid if applicable)
* Improving UI/UX through spacing, typography, and alignment
* Organizing files for a clean project structure
